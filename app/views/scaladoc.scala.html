Scalactic's Scaladoc documentation

Scalactic is released as part of the ScalaTest project. For every release of ScalaTest, there is a corresponding release of Scalactic with the same version number. Thus if you use both Scalactic and ScalaTest, you should make sure the version numbers match.

Scalactic Scaladoc documentation is integrated with ScalaTest's documentation. The links on this page link to the Scalactic portion of ScalaTest's Scaladoc documentation:

  • Scaladoc for Scalactic 2.1.0 (the latest release for Scala 2.10.0+)
  • Scaladoc for Scalactic 2.0 (for Scala 2.10.0+)
